# T05: Accés Remot – Connexió via SSH (Tasca Individual)

## Breu Descripció
Aquesta tasca tracta una de les operacions més crítiques de la nostra consultora: **gestió remota segura**. Els servidors dels clients i els nostres propis servidors es troben en CPDs o al núvol, per la qual cosa l’accés físic és l’excepció i no la norma. La nostra eina principal és **SSH (Secure Shell)**, l’estàndard de la indústria per administrar màquines Linux de manera xifrada i eficient.

## La Teva Missió
Crear una **Base de Coneixement interna** per als futurs becaris. Hauràs de generar una **Prova de Concepte (PoC)** documentada que servirà de guia interna, mostrant com connectar-se als nostres sistemes de manera segura.

### Objectius:
- Documentar la connexió SSH entre els nostres sistemes.
- Fer servir **màquines virtuals** per simular els entorns.
- Instruir sobre connexions des de:
  - **Clients Linux**: Terminal nativa.
  - **Clients Windows**: PowerShell o terminals moderns.
