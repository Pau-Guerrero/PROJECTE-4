# Accés remot – Escriptori remot (RDP)

## Descripció de la tasca
En aquesta tasca individual, l’objectiu és documentar i crear una prova de concepte (PoC) per al suport remot als usuaris finals. 

Com a consultors d’EverPia, no només gestionem servidors via SSH, sinó que també hem de poder donar suport gràfic a clients quan informen d’errors o problemes amb aplicacions. Això requereix establir connexions d’escriptori remot amb accés complet a la pantalla, ratolí i teclat del client.

La tasca se centra en:

- **RDP (Remote Desktop Protocol):** l’estàndard de Microsoft per a Windows, que també és compatible amb alguns entorns GNU/Linux com Zorin OS amb escriptoris gràfics Gnome.
- Creació de guies clares i detallades per als futurs becaris sobre com establir connexions d’escriptori remot.
- Prova de concepte interna utilitzant màquines virtuals per simular connexions reals i documentar el procés.

L’objectiu és que la documentació sigui fàcil de seguir i permeti resoldre incidències de manera ràpida i eficient.
