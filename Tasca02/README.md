# DPR: Còpies de seguretat – Cas pràctic

## Descripció de la tasca
Aquest projecte té com a objectiu documentar i provar un pla de còpies de seguretat per a l’empresa **Muntatges i Serveis Tècnics SL**. La tasca consisteix a elaborar guies tècniques i proves de concepte per garantir que tant els equips Windows dels usuaris com els servidors Linux tinguin còpies de seguretat fiables i restaurables.

S’han definit dues parts principals:

1. **Còpia de seguretat equips Windows**  
   - Configurar una màquina virtual amb Windows 11.  
   - Crear còpies de seguretat locals a un disc secundari i al cloud (Google Drive) utilitzant **Duplicati**.  
   - Documentar el procés d’instal·lació, configuració i restauració de fitxers.

2. **Còpia de seguretat servidor Linux**  
   - Utilitzar una màquina virtual amb Ubuntu Server i un disc de backup adicional.  
   - Configurar **Duplicity** per fer còpies completes i incrementals de la carpeta `/home`.  
   - Automatitzar els backups amb scripts i cron.  
   - Documentar el procés d’instal·lació, creació d’usuaris, creació de fitxers de prova, còpies i restauracions.

L’objectiu final és generar una guia tècnica clara que permeti al personal de l’empresa implantar i mantenir les còpies de seguretat de forma segura.
